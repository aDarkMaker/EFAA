## 2. 快速开始

1. **下载安装包**：从下方的“下载方式”中选择一个，下载最新的 `EFAA-Chinese-setup.exe` 安装文件。
2. **安装程序**：双击 `EFAA-Chinese-setup.exe` 文件，并按照安装向导的提示完成安装。
3. **运行程序**：安装完成后，从桌面快捷方式或开始菜单启动 `EFAA` 即可。

## 4. 功能介绍

- **运行环境**：
  - 支持 ADB 功能
  - 推荐使用 MuMu 模拟器
  - 规格：1920 x 1080，280 DPI
- **支持功能**：
  - 帝江号基建收菜
  - 地图已解锁资源点日常回收
  - 地图已解锁贵重资源采集
  - 高难淤积点 / 日常资源副本体力消耗
  - 每日商城信赖收取 + 兑换物资
  - 殖民地日常资源交换 + 自动炒股
