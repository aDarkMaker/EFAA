# JS
- do NOT import React from 'react'. Use the latest React features.
- prefer the unpack operator `{ ... }` to import things from libraries.
- prefer the alias `@/` for importing things from the src directory.
- prefer Function Components over Class Components.
- use `import type` instead of `import` for type imports.

# Styles
- prefer the tailwindcss classes.
- prefer the variables from the @theme section of the index.css file.

# lint
- 2 space indentation.
- standard js style.
- use single quotes for strings.
- simply lint the code with eslint fix command.
